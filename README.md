# 🛒 E-Commerce Frontend (Vue 3)

A modern and responsive e-commerce frontend application built using Vue 3. This project demonstrates core frontend development concepts like component-based architecture, state management, API integration, and routing.

---

## 🚀 Features

* 🏠 Home page displaying products
* 📄 Product detail page
* 🛒 Shopping cart functionality
* ➕ Add / remove items from cart
* 🔄 Dynamic data fetched from external e-commerce API
* ⚡ Loading indicators for better UX
* 📱 Fully responsive design using Bootstrap

---

## 🛠️ Tech Stack

* **Frontend Framework:** Vue 3
* **Styling:** Bootstrap
* **State Management:** Vue Store (Cart handling)
* **Routing:** Vue Router
* **API Integration:** External E-commerce API
* **Build Tool:** Vite

---

## 📂 Project Structure

```
src/

├── views/            # Page-level components
│   ├── HomeView.vue
│   ├── ProductDetailView.vue
│   └── CartView.vue
│

│   └── cart.js
│

│   └── api.js
│
├── routers/          # Routing configuration
│   └── index.js
│
└── main.js           # App entry point
```

## ⚙️ Installation & Setup

1. Clone the repository:

```bash
git clone https://github.com/your-username/ecommerce-frontend.git
```

2. Navigate into the project:

```bash
cd ecommerce-frontend
```

3. Install dependencies:

```bash
npm install
```

4. Run the development server:

```bash
npm run dev
```
